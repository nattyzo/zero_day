<h2>Vagrant</h2>
<p>Vagrant is an open-source software product for building and maintaining portable virtual software development environments.
Vagrant is a tool for building and managing virtual machine environments in a single workflow. With an easy-to-use workflow and 
focus on automation, Vagrant lowers development environment setup time, increases production parity, and makes the "works on my 
machine" excuse a relic of the past.i</p>
